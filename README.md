# 3E---ETEC---Gabaritando
Plataforma escolar baseada na meritocracia que organiza rankings por turma, companhia e geral, destacando e premiando os melhores alunos para incentivar uma competição saudável. Também permite o acompanhamento das notas em tempo real, com alertas e recomendações de reforço, ajudando no engajamento e na melhoria contínua do desempenho acadêmico.
